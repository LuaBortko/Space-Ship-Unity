# Space-Invaders-Unity
Projeto desenvolvido na Unity como atividade de laboratório, consistindo na criação de um jogo simples de nave, contendo uma mecânica de desacelerar o tempo.

## Requisitos do Projeto
- Movimentação  
- Pontuação
- Tiros da Nave
- Capacidade de desacelerar o tempo
- Tela de derrota
- Tela de vitória
